# Quote Slider

A simple quote slider built with vanilla JavaScript. Features smooth animations and navigation between inspirational quotes.

## Features
- Next/Previous navigation
- Infinite loop
- Smooth animations  
- Responsive design

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)

## Usage
Click "Next" or "Back" buttons to navigate through quotes.
